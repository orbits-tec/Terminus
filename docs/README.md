# Terminus

O Terminus é um emulador de terminal para macOS baseado na biblioteca [SwiftTerm](https://github.com/migueldeicaza/SwiftTerm).

### Pre-requisitos

- macOS 15 ou superior

### Recursos

- Emulação Xterm
- Múltiplas abas
- Split terminal
- Cadastro de sessões SSH

### Atalhos

- ` ⌘D  ` dive o terminal ativo verticalmente
- ` ⌘⇧D ` divide o terminal ativo horizontalmente
- ` ⌘⌃W ` fecha o painel ativo
